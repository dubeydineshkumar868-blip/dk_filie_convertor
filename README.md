# 🔄 DK File Converter

Your all-in-one file conversion toolkit built with Streamlit.

## ✨ Features

- 🖼️ **Image to PDF**: Combine multiple images into a single PDF file
- 🖼️ **Image to Word**: Insert images into a well-formatted Word document
- 📄 **PDF to Image**: Extract each page of a PDF as high-quality PNG images
- 📝 **PDF to Word**: Convert PDFs to editable Word documents
- 📊 **PDF to Excel**: Extract tables from PDFs into Excel spreadsheets
- 📎 **Merge PDFs**: Combine multiple PDF files into one

## 🚀 Run Locally

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## 🛠️ Built With

- [Streamlit](https://streamlit.io/) - The web framework
- [Pillow](https://python-pillow.org/) - Image processing
- [PyMuPDF](https://pymupdf.readthedocs.io/) - PDF manipulation
- [pdf2docx](https://pypi.org/project/pdf2docx/) - PDF to Word conversion
- [pdfplumber](https://github.com/jsvine/pdfplumber) - Table extraction from PDFs
- [pandas](https://pandas.pydata.org/) - Data analysis for Excel conversion
- [openpyxl](https://openpyxl.readthedocs.io/) - Excel file handling
